### 2.2 因子挖掘方向

#### 学习目标
- 理解因子分类体系（价值因子、动量因子、质量因子、情绪因子）
- 掌握因子有效性检验方法（IC、IR、分位数回测）
- 学习因子组合与权重优化

#### 目标项目清单
| 项目 | Stars | 特点 | 优先级 |
|------|-------|------|--------|
| **qlib** | 40k+ | 完整的因子挖掘框架 | P0 |
| **backtrader** | 13.5k | 包含基础因子库 | P1 |
| **scikit-learn** | 56k+ | 机器学习因子构建 | P1 |
| **TA-Lib** | 10.5k | 技术指标因子库 | P0 |

#### 因子分类体系
1. **价值因子**：
   - PE、PB、PS、PEG
   - 股息率
   - EV/EBITDA
   - 市净率

2. **动量因子**：
   - 相对强弱指标 (RSI)
   - MACD
   - 价格动量 (Momentum)
   - 累计收益率

3. **质量因子**：
   - ROE、ROA
   - 负债率
   - 现金流
   - 毛利率

4. **情绪因子**：
   - 换手率
   - 成交量变化
   - 新闻情绪
   - 社交媒体情绪

5. **规模因子**：
   - 市值
   - 流通市值
   - 自由流通市值

#### 因子有效性检验
1. **IC (Information Coefficient)**：
   - 因子值与未来收益的相关系数
   - IC > 0.03 为有效因子

2. **IR (Information Ratio)**：
   - IC的均值除以IC的标准差
   - IR > 0.5 为稳定因子

3. **分位数回测**：
   - 将股票按因子值分为5组
   - 计算各组收益率
   - 最优组应该显著跑赢最差组

### 2.4 风险管理方向

#### 学习目标
- 掌握仓位管理策略（凯利公式、固定比例）
- 学习止损止盈设置（ATR止损、百分比止损）
- 理解风险控制指标（VaR、CVaR）

#### 目标项目清单
| 项目 | Stars | 特点 | 优先级 |
|------|-------|------|--------|
| **zipline** | 19.5k | 内置风险管理模块 | P0 |
| **QuantConnect** | 7.8k | 完整的风控体系 | P1 |
| **QuantLib** | 3.8k | 定量金融库 | P2 |

#### 仓位管理策略
1. **固定比例**：
   - 每只股票固定仓位
   - 简单但可能不够灵活

2. **凯利公式**：
   ```
   f* = (bp - q) / b
   f* = 最优仓位比例
   b = 赔率
   p = 胜率
   q = 败率 (1 - p)
   ```

3. **波动率调整**：
   - 高波动低仓位
   - 低波动高仓位
   - 动态调整

#### 止损止盈策略
1. **ATR止损**：
   - 基于 Average True Range
   - 止损距离 = N * ATR
   - 自适应波动

2. **百分比止损**：
   - 固定百分比（如5%）
   - 简单易用

3. **移动止损**：
   - 追踪价格变动
   - 保护利润

#### 风险控制指标
1. **VaR (Value at Risk)**：
   - 在一定置信度下的最大可能损失
   - 如：95%置信度下的1日VaR

2. **CVaR (Conditional VaR)**：
   - 超过VaR时的平均损失
   - 也叫 Expected Shortfall

3. **最大回撤**：
   - 从峰值到谷底的最大跌幅
   - 控制在可接受范围内

## 第三阶段：实战应用与整合

### 3.1 选股能力提升方案

#### 技术选股（40%权重）
- 技术指标：MA、MACD、RSI、KDJ、BOLL、ATR
- 价格形态：突破、回踩、反转、整理
- 趋势判断：上升趋势、下降趋势、横盘震荡

#### 基本面选股（30%权重）
- 财务指标：PE、PB、ROE、ROA、毛利率
- 成长性：营收增速、净利润增速
- 估值水平：历史分位数、行业比较

#### 资金面选股（30%权重）
- 主力资金流向
- 机构持仓变化
- 北向资金动向
- 换手率、成交量

#### 综合评分体系
```
总分 = 技术面*0.4 + 基本面*0.3 + 资金面*0.3
总分 > 70：强烈买入
60 ≤ 总分 < 70：买入
50 ≤ 总分 < 60：持有
40 ≤ 总分 < 50：观望
总分 < 40：卖出
```

### 3.2 买卖点判断方案

#### 买入信号（满足3个以上）
1. **技术买入信号**：
   - 金叉（短期均线上穿长期均线）
   - 突破关键阻力位
   - RSI从超卖区回升
   - 成交量放大

2. **资金买入信号**：
   - 主力资金连续流入
   - 机构加仓
   - 北向资金大额买入

3. **基本面买入信号**：
   - 财报超预期
   - 业绩拐点确认
   - 行业景气度提升

#### 卖出信号（满足3个以上）
1. **技术卖出信号**：
   - 死叉（短期均线下穿长期均线）
   - 跌破关键支撑位
   - RSI进入超买区
   - 高位放量滞涨

2. **资金卖出信号**：
   - 主力资金连续流出
   - 机构减仓
   - 北向资金大额卖出

3. **基本面卖出信号**：
   - 财报不及预期
   - 业绩拐点向下
   - 行业景气度下降

#### 止损止盈设置
- **止损**：
  - ATR止损：价格 - 2*ATR
  - 百分比止损：成本价 * 0.95
  - 技术止损：跌破关键支撑位

- **止盈**：
  - 第一目标：+10%
  - 第二目标：+20%
  - 第三目标：+30%
  - 移动止盈：最高价回落5%时止盈
