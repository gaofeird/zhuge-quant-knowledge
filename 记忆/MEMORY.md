## 量化交易核心知识（2026年4月18日学习整理）

### 核心认知铁律
1. **量化交易的核心不是"策略"本身，而是开发策略的能力**
2. **没有永远有效的"圣杯因子"**，市场风格轮动导致因子效用周期性变化
3. **90%的回测结果可能存在幻觉**，回测优秀 ≠ 实盘有效
4. **多因子策略本质是分散单一因子失效风险**

### 回测四大陷阱
1. 过拟合：参数过度优化
2. 未来函数：使用当时未知信息
3. 幸存者偏差：只考虑存活股票
4. 交易成本忽视：滑点、手续费、冲击成本

### 多因子选股原则
- **"重因子轻模型"**，避免过度复杂设计
- 简单有效：大市值 + 高分红 + 低PE
- 因子组合分散风险

### 风控铁律
1. 个股短期持有，盈利及时离场
2. 避免反复操作同一标的
3. 单策略仓位不超过30%
4. 必须设置止损止盈

### 实盘验证策略案例
- 可转债1.1进攻策略：年化72.2%，回撤11.92%，夏普2.852
- 特点：回撤控制优秀，换手率适中（18.17%）
