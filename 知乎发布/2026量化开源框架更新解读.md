## 一、TradingAgents：多智能体框架里程碑升级

**⭐ Stars：46k+**

**最新版本**：v0.2.3（2026年3月）

### 核心更新

| 更新项 | 说明 |
|-------|------|
| 🌍 **多语言支持** | CLI支持**11种语言**输出，分析报告本地化 |
| 🤖 **GPT-5.4系列** | 支持GPT-5.4 Mini/Nano，默认模型升级 |
| 📊 **回测保真度** | 日期感知的数据获取，避免前视偏差 |
| 🔌 **代理支持** | Google/Anthropic客户端支持代理 |
| ⭐ **五级评级** | Buy/Overweight/Hold/Underweight/Sell |

### 多智能体架构

TradingAgents模拟真实交易公司的运作：

```
分析师团队
├── 基本面分析师：财务数据、估值分析
├── 情绪分析师：市场情绪、资金流向
├── 新闻分析师：舆情监控、事件驱动
└── 技术分析师：均线、MACD、RSI

        ↓

研究团队
├── 多头研究员：看涨论点
└── 空头研究员：看跌论点（对抗辩论）

        ↓

决策层
├── 交易员：执行决策
├── 风控团队：风险评估
└── 投资组合经理：最终批准
```

**创新亮点**：
- **辩论机制**：多头vs空头对抗辩论，确保决策客观
- **五级评级**：对标专业机构的评级体系
- **回测保真**：杜绝"偷看未来"的前视偏差

### 快速上手

```bash
pip install tradingagents
tradingagents

# 工作流程：
# 1. 选择股票代码
# 2. 设置分析日期
# 3. 选择LLM提供商和输出语言
# 4. 运行分析
```

## 二、Qlib：微软量化平台新增情绪因子

**⭐ Stars：40k+**

**最新更新**：2026年4月

### 美国零售情绪数据集

Qlib新增基于Adanos的零售情绪数据，覆盖**四大来源**：

| 来源 | 核心字段 |
|------|---------|
| Reddit | reddit_buzz, reddit_sentiment, reddit_mentions |
| X(Twitter) | x_buzz, x_sentiment, x_mentions |
| News | news_buzz, news_sentiment, news_mentions |
| Polymarket | polymarket_buzz, polymarket_sentiment |

### Alpha158AdanosUS因子

在Alpha158基础上，增加**滞后情绪因子**：

```python
# 核心情绪特征
lagged_retail_buzz          # 滞后零售热度
lagged_retail_sentiment     # 滞后零售情绪
disagreement_ratio          # 分歧比率
```

⚠️ **重要设计**：所有情绪因子都经过**滞后处理**，避免当日数据泄露导致的过拟合。

### 其他更新

- 🇬🇧 伦敦证券交易所支持
- 📈 风险指标增强：Sortino、Calmar、最大回撤持续期

## 三、FinRL-X：强化学习实盘验证

**⭐ Stars：14k+**

**最新发布**：FinRL-X

### 架构设计

```
数据层 → 策略模块 → 回测引擎 → 执行层
              ↓
       权重中心化接口
```

**核心理念**：策略即插即用，统一执行语义。

### 实盘模拟验证（2025.10 - 2026.03）

| 指标 | FinRL-X | SPY基准 | QQQ基准 |
|------|---------|---------|---------|
| 累计收益 | **+19.76%** | -2.51% | -4.79% |
| 年化收益 | **62.16%** | -6.60% | -12.32% |
| 夏普比率 | **1.96** | -0.55 | -0.73 |
| 最大回撤 | -12.22% | -5.35% | -7.88% |
| 胜率 | **64.89%** | 52.13% | 54.02% |

> 📌 说明：短期数据不足以建立统计显著性，但验证了端到端执行流程的稳定性。
